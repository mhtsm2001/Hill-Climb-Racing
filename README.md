
# Hill Climb Racing
This website was created to provide information about the popular game **Hill Climb Racing**.

## About the Project
The site is designed as an informational page where visitors can explore different parts of the game, including:

- 🚗 **All Vehicles** available in the game
- 🗺️ **Maps / Stages** with images and basic information
- ⭐ **Special Versions** and unique content related to the game

The goal of the project is to present the game's content in a clean and simple visual format.

## Images & Media
The website includes a collection of images showcasing:
- All vehicles
- Game maps
- Special editions of the game

These images help visitors quickly explore the game's content.

## Demo Store
The website also includes a **simple demo store** where users can browse and "buy" game-related products.

⚠️ Important:  
This store is **not real** and **no real payments are processed**.  
It is only a **visual and educational feature** created to demonstrate how an online shop interface works.

## Purpose
This project was built for:
- Practice in **web design**
- Learning **front-end development**
- Creating a **game information website**
